# 💰 Personal Finance Tracker

A full-stack **MERN Stack Personal Finance Tracker** that helps users manage income, expenses, and financial activities with a clean and responsive dashboard.

## 🚀 Live Demo

Coming soon...

## 📌 Features

✅ User Authentication (Register & Login)  
✅ JWT Based Protected Routes  
✅ Add Income and Expense Transactions  
✅ Edit Transactions  
✅ Delete Transactions  
✅ Search Transactions  
✅ Filter Transactions by Category  
✅ Filter Income / Expense  
✅ Dashboard with Balance Summary  
✅ Income and Expense Analytics  
✅ Interactive Charts  
✅ Responsive Design for Mobile and Desktop  

---

## 🛠️ Tech Stack

### Frontend

- React.js
- Tailwind CSS
- React Router DOM
- Axios
- JavaScript ES6+

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
